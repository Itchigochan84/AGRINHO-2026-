# 🌍 Projeto Agrinho 2026

## 📝 Protegendo a Vida da Natureza e da Agropecuária

Este site foi desenvolvido como parte do projeto para o **Concurso Agrinho**, com o objetivo de debater o equilíbrio fundamental entre o crescimento da produtividade na agropecuária e a preservação do meio ambiente.

O projeto utiliza como base os **Objetivos de Desenvolvimento Sustentável (ODS) da Agenda 2030** da Organização das Nações Unidas (ONU) para propor reflexões sobre como garantir a segurança alimentar e a economia global, sem degradar os recursos naturais necessários para as próximas gerações.

---

## 🚀 ODS Abordadas no Trabalho
O site detalha o impacto e a importância de cinco objetivos principais para o futuro do campo:
*   **ODS 2:** Fome Zero e Agricultura Sustentável
*   **ODS 6:** Água Potável e Saneamento
*   **ODS 12:** Consumo e Produção Responsáveis
*   **ODS 13:** Ação Contra a Mudança Global do Clima
*   **ODS 15:** Vida Terrestre

---

## 🛠️ Tecnologias Utilizadas
Para a construção deste site de forma organizada e limpa, utilizamos:
*   **HTML5:** Para a estruturação semântica de todos os textos e imagens.
*   **CSS3:** Para a estilização visual, garantindo um design responsivo (adaptado para celulares).
*   **JavaScript:** Para adicionar elementos de interatividade e reflexões dinâmicas para o usuário.
*   **GitHub Pages:** Para a hospedagem pública e distribuição do site na internet.
